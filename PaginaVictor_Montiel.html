En esta pagina voy a aprovechar para darles un repaso a las propiedades y terminos basicos de CSS.

Lista de terminos que podran ayudarte a enteder mejor como estilizar una pagina web:

1. color: Define el color del texto. color: blue;

2. font-size: Define el tamaño de la fuente. font-size: 16px;

3. font-weight: Define el grosor de la fuente (normal, bold, etc.). font-weight: bold;

4. border: Define el borde alrededor de un elemento, con grosor, estilo y color. border: 1px solid black;

5. transform: Aplica transformaciones como rotación, escalado, traslación, etc. transform: rotate(45deg);

6. background-image: Establece una o más imágenes de fondo para un elemento. url | none | initial | inherit;

7. font-family: Define la fuente del texto. font-family: Arial, sans-serif;

8. margin: Establece el espacio exterior de un elemento. margin: 10px;

9. top, right, bottom ó left: Define la posición exacta de un elemento cuando se usa con position. top: 10px; left: 20px;

10. opacity: Define la transparencia de un elemento (0 es completamente transparente, 1 es opaco). opacity:
